# Briefings

Generated daily job search intelligence reports.

Each morning (7:00-7:30 AM), the system runs three briefing parts:
1. **Roles**: New matching roles + scoring + auto-tailor triggers
2. **Networking**: New connections + follow-up nudges + referral opportunities
3. **Coaching**: Pipeline summary + bottleneck detection + daily priority stack

See `../cowork-tasks/daily-morning-briefing.md` for how to activate.

---

## File Structure

Each briefing is timestamped:
- `briefing-2026-05-07.md` — Full morning briefing (all 3 parts)
- `briefing-2026-05-07-part1-roles.md` — Roles only
- `briefing-2026-05-07-part2-networking.md` — Networking only
- `briefing-2026-05-07-part3-coaching.md` — Coaching only

---

## What to Do With Them

1. **On Wake**: Read the full briefing (3-4 min)
2. **Action Items**: Open the links, mark yes/no on roles, respond to network nudges
3. **Weekly**: Friday briefing summarizes the week (pattern spotting, adjustments)
4. **Archive**: Keep for pattern analysis later

---

## Notes

- Briefings are personalized to your `career-plan.md`, `target-companies.md`, and `interview-history.md`
- If your target parameters change, regenerate the previous day's briefing to see what you missed
- Briefings work best when your context files are up-to-date (role tracker, networking map, etc.)
